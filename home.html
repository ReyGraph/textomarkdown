Markdown Editor Guide

  • Bold text: Enclose text with ** or __
  • Italic text: Enclose text with * or _
  • Heading 1: Prefix text with # (e.g., # My Heading)
  • Heading 2: Prefix text with ## (e.g., ## Subheading)
  • Heading 3: Prefix text with ### (e.g., ### Sub-subheading)
  • Bulleted list: Prefix each item with - (e.g., - Item 1)
  • Numbered list: Prefix each item with 1. (e.g., 1. Item A)
  • Inline code: Enclose text with ` (backtick)
  • Insert an emoji by typing `:emoji-name:`
